# Build AI With Us: Full Curriculum Overview

Welcome to the **Build AI With Us** curriculum catalog. This document outlines the structured learning path designed for kids and teens (ages 10+), transitioning them from absolute beginners (curious users of AI) into technical builders who ship real-world apps, agents, and systems.

---

## 🚀 The Core Program (Age 10+)
*10 progressive modules — from zero to building and deploying real AI applications.*

### Module 01: Introduction to AI & Machine Learning (This Repository)
*   **Focus**: What AI actually is, how it thinks, and why it matters right now.
*   **Key Concept**: Rules-based coding vs. learning from data (Machine Learning).
*   **Activity**: Interactive classification games and training a simple brain.

### Module 02: Types of AI & ML
*   **Focus**: Supervised learning, unsupervised learning, and generative AI demystified.
*   **Key Concept**: How machines learn from labeled images, find hidden patterns, and create new content.
*   **Activity**: Interactive grouping challenges and generative AI prompt experiments.

### Module 03: Prompt Engineering
*   **Focus**: The art and science of communicating with Large Language Models (LLMs) to get high-quality results.
*   **Key Concept**: Context, instructions, persona, and output formatting.
*   **Activity**: Building a "Choose Your Own Adventure" story game using structured system instructions.

### Module 04: AI Chatbot with n8n
*   **Focus**: Introduction to visual automation and connecting AI to the real world.
*   **Key Concept**: API endpoints, webhook triggers, node-based programming.
*   **Project**: A live chat assistant with a custom personality connected to Discord or Telegram.

### Module 05: Voice Agent with Retell AI
*   **Focus**: Building conversational voice bots that sound and react like humans.
*   **Key Concept**: Text-to-Speech (TTS), Speech-to-Text (STT), and low-latency response models.
*   **Project**: A smart phone receptionist that answers calls and responds in real time.

### Module 06: Computer Vision & Smart Cameras
*   **Focus**: Teaching computers to "see" and recognize the physical world.
*   **Key Concept**: Image classification, dataset collection, training epochs, and confidence scores.
*   **Project**: Training a Google Teachable Machine model to control a Scratch game via hand gestures or identify objects through a webcam.

### Module 07: Introduction to Web Coding with AI
*   **Focus**: Learning the fundamentals of web user interfaces (HTML, CSS, basic Javascript) using AI coding assistants.
*   **Key Concept**: Document Object Model (DOM), visual design systems, and developer tools.
*   **Project**: Building a personal portfolio landing page with interactive elements.

### Module 08: Integrating AI into Web Apps
*   **Focus**: Connecting web frontends to AI brains.
*   **Key Concept**: Fetching data from APIs, handling async requests in JavaScript, and managing API keys securely.
*   **Project**: A functional web-based AI Study Assistant that summarizes notes or quizzes students on their homework.

### Module 09: Automated Booking & Business Systems
*   **Focus**: Setting up professional AI automations for real businesses.
*   **Key Concept**: Calendar APIs, databases (Airtable/Supabase), and email triggers.
*   **Project**: An end-to-end appointment booking agent that answers text messages, finds available slots, and schedules them.

### Module 10: Capstone Project: My First AI Startup
*   **Focus**: Combining all previous concepts (Web UI + n8n automation + LLM brains) into a complete product.
*   **Key Concept**: Full-stack application lifecycle, product design, and deployment.
*   **Project**: Launching a complete AI SaaS product (e.g., an automated blog post creator, custom workout planner, or meal assistant).

## ⚡ The Advanced Program (After Core)
*5 expert tracks — build and deploy production-grade AI systems.*

### Module 11: Automation for CRMs & ERPs
*   **Focus**: Building heavy enterprise workflows.
*   **Skills**: Webhooks, JSON parsing, error handling, rate limiting, and connecting Hubspot/Salesforce/Slack.

### Module 12: Development with Claude Code / Cursor
*   **Focus**: Developing full-stack apps at speed using AI IDEs and command-line AI tools.
*   **Skills**: Directing AI as a "Senior Developer," debugging, git integration, and multi-file code editing.

### Module 13: Multi-Agent Orchestration
*   **Focus**: Designing systems where specialized AI agents collaborate to solve complex tasks.
*   **Skills**: LangChain/CrewAI, agent roles, message passing, loops, and autonomous task execution.

### Module 14: Custom ML Models with Python
*   **Focus**: Getting under the hood with Python and machine learning libraries.
*   **Skills**: Pandas, NumPy, training custom classifiers, fine-tuning open-source models (Llama/Hugging Face).

### Module 15: Deploy to Production
*   **Focus**: Launching applications to the live internet.
*   **Skills**: Virtual Private Servers (VPS), Docker, SSL certificates, GitHub Actions, and custom domains.
